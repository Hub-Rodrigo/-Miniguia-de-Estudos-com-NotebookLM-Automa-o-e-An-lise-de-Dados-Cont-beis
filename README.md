# 📊 Caderno Temático com IA: Análise de Dados Contábeis

## 🎯 Contexto

Este projeto foi desenvolvido como parte de um desafio prático com o objetivo de utilizar Inteligência Artificial como ferramenta de aprendizado ativo.

O foco foi aplicar conceitos de análise de dados no contexto contábil, buscando otimizar rotinas operacionais e melhorar a qualidade das informações utilizadas na tomada de decisão.

---

## 🚀 Objetivos

* Aplicar IA como apoio no aprendizado técnico
* Melhorar a organização de dados contábeis
* Estruturar análises no Excel e Power BI
* Desenvolver prompts mais eficientes
* Criar um material reutilizável (miniguia)

* Falta de contexto nos prompts iniciais
* Respostas muito genéricas

📌 Solução:
Uso de prompts mais específicos e orientados a cenário real.

## 🧠 Miniguia de Estudo

### 📌 Resumo

A análise de dados contábeis depende de três pilares:

1. Organização dos dados (ETL)
2. Modelagem de dados (fato e dimensão)
3. Visualização (dashboards)

Modelo ideal: ⭐ Modelo Estrela

---

### 📖 Glossário

* ETL: Extração, Transformação e Carga
* Tabela Fato: onde ficam os valores
* Tabela Dimensão: contexto das análises
* DAX: linguagem do Power BI
* CFOP: Código Fiscal de Operações

---

### 🔁 Prompts Reutilizáveis

* Explique [conceito] com exemplo contábil
* Crie um modelo de dados para [problema]
* Quais erros comuns em [processo]?
* Como automatizar [processo] no Excel?
